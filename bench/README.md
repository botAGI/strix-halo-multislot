# Харнессы

Python 3.10+, единственная зависимость `requests` (`pip install -r requirements.txt`).
Целевой сервер задаётся переменной окружения, по умолчанию `http://localhost:8080`.
Клиент гоняйте с СОСЕДНЕЙ машины, не с тестируемой.

## bench_slots.py — мультислот-свип генерации

Уровни конкурентности через запятую; окно 75с на уровень; уникальные salted-промпты,
`cache_prompt: false`; печатает агрегат, скорость на запрос p50 и серверную
обработку промпта.

```bash
BENCH_BASE=http://host:8080 python bench_slots.py 1,8,16,32
N_PREDICT=512 BENCH_BASE=http://host:8080 python bench_slots.py 8,32   # длинные генерации
```

## bench_rag_stream.py — длинные промпты со СТРИМИНГОМ

Промпты ~3,4К токенов, ответ 128; меряет настоящий клиентский TTFT (метка первого
чанка, очередь и сеть входят), серверную обработку промпта и полное время ответа.

```bash
BENCH_BASE=http://host:8080 python bench_rag_stream.py 1,4,8,16
```

## bench_rag.py — то же без стриминга (историческая версия)

Оставлен для воспроизведения ранних таблиц; вместо клиентского TTFT печатает
только серверный prompt_ms. Для новых замеров берите стриминговую версию.

## bench_sustained.py — выносливость

Непрерывная нагрузка 32 клиентами N минут, поминутные бакеты (по минуте
ЗАВЕРШЕНИЯ запроса — учитывайте артефакт на границах), итоговый средний агрегат.
Телеметрию GPU (температура/мощность/частоты) снимайте параллельным циклом
по sysfs: `/sys/class/hwmon/hwmon*/temp1_input`, `power1_average`,
`/sys/class/drm/card*/device/pp_dpm_sclk`.

```bash
BENCH_BASE=http://host:8080 python bench_sustained.py 30
```

## bench_er.py — embed / rerank

Пейлоад из JSON-файла, уровни конкурентности, окно 45с; печатает rps и латентности.
Примеры пейлоадов лежат в `payloads/` (emb1/emb32/emb64/emb128 — эмбеддинги поштучно и пачками, rr10/rr20/rr40 — реранк с разным числом документов). Формат: `{"input": "текст"}` или `{"input": [...]}` для
эмбеддингов, `{"query": "...", "documents": [...]}` для реранка.

```bash
BENCH_URL=http://host:8081/v1/embeddings python bench_er.py payloads/emb1.json 4,16,32
BENCH_URL=http://host:8082/v1/rerank     python bench_er.py payloads/rr20.json 1,4,8
```

## Канон замеров (использованный в results/)

Скрининг 1×75с; финальные числа 3 прогона, медиана; агрегат = Σ predicted_n /
makespan по серверным `timings`; провалы публикуются. Эффективную конфигурацию
сервера сверяйте по `GET /props` (`total_slots`), а не по списку флагов.
